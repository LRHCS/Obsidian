# Aufbau
1. Seil(Zuständig für die Kraftübertragung)
	- Zugkraft($F_Z$)
	- Gewichtskraft($F_L$)
2. Rollen
	- Feste Rolle
		- Zugweg($S_Z$) = Zürückgelegte Weg($S_L$)
	- Lose Rolle
		- $F_Z =\frac {1}{2}*F_L$ 
		-  $S_Z =2*S_L$ 
# Funktion
$F_Z <-> F_n$
- n = Tragende Seil
	- jede Seilstück dass zwischen eine Feste Rolle und eine Lose Rolle verläuft
- $F_Z =\frac {1}{n}*F_L =\frac {1}{n}*m*g$ 
- $S_Z = n * S_L$

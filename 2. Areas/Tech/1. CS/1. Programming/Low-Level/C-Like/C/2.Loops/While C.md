 #C #loops   
## Syntax
```C
while (_condition_) {  
  _// code block to be executed_  
}
```

## Example
```C
int i = 0;  
  
while (i < 5) {  
  printf("%d\n", i);  
  i++;  
}
```

## Do-While
With do-while, they are going to execute the code in do once before they check the conditions
```C
int i = 0;  
  
do {  
  printf("%d\n", i);  
  i++;  
}  
while (i < 5);
```

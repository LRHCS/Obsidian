 #java #loops    
### Syntax

```java
while (condition) {
  // code block to be executed
}
```

### Example

```java
int i = 0;
while (i < 5) {
  System.out.println(i);
  i++;
}
```

## The Do/While Loop

### Syntax

```java
do {
  // code block to be executed
}
while (condition);
```

### Example

```java
int i = 0;do {
  System.out.println(i);
  i++;
}
while (i < 5);
```

  #C #if_else   

## Syntax
```C
if (_condition_) {  
  _// block of code to be executed if the condition is true_  
}
```

## Example

```C
#include <stdio.h>  

int main() {  

    int i = 0;

  

    if(i == 10){

        printf("i is 10");

    } else if(i < 10){

        printf("i is smaller than 10");

    } else {

        printf("i is smaller than 10");

    }

}
```

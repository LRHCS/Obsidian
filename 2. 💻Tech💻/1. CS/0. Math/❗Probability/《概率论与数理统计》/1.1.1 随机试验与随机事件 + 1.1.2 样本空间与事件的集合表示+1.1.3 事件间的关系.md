# 1.1.1 随机试验与随机事件
确定性（必然）一定发生（不发生）确定现象
随机（偶然）可能发生可能不发生 随机现象

统计规律 做大量实验发现规律

## 随机事件
### 实验:
观察，测量，科学实验
### 随机实验:
- 可重复性-相同的条件下可以重复
- 实验结果不止一个
- 无法预测那个结果
用E表示

### 事件:
- 每一种结果都是一个事件
随机事件
- 可能发生可能不发生

### 基本事件
- 相对于实验目的来说不能再分，不必再分
	- 硬币正反
	- 色子 1-6点
### 复合事件
- 有基本事件复合而成
	- 色子小于5点
		- 包括1-4
		- Ω = 样本空间 权集
	- 色子大于7点
		- Φ = 空集

### 不随机
- 极端
#### 必然事件
- 每次实验必然发生的结果 Ω

#### 不可能事件
- 不可能发生的事件 Φ

# 1.1.2 样本空间与事件的集合表示
## 样本空间 Ω
定义: 所有基本事件的集合

## 样本点: 
样本空间的元素，基本事件 ω

色子 : Ω = {1，2，3，4，5，6}
两个硬币 : Ω = {正正，反反，正反，反正}
电话交换台 Ω = {n} 无限集

例子：
a，b 扔一质子，求坐标
Ω = {a，b}
像平面扔一质子 
Ω = {（x，y）| x，y = R}
像空间扔一质子
Ω = {（x，y，z）| x，y，z = R}

事件用集合表示
A = {2，4，6}  C= {1，2}

Ω= 必然事件=样本空间
Φ=不可能事件=空集
事件 Ω的子集

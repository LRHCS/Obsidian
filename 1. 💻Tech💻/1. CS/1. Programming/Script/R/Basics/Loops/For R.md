## For
the for loop is very similar as the for loops in python
```R
for (x in 1:10) {  
  print(x)  
}
```
You can also print the word in a list or array
```R
fruits <- list("apple", "banana", "cherry")  
  
for (x in fruits) {  
  print(x)  
}
```
```R
dice <- c(1, 2, 3, 4, 5, 6)  
  
for (x in dice) {  
  print(x)  
}
```

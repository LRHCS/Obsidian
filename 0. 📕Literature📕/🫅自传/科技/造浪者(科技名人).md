# 利普·卡兹
- ".zip"
- 第一行是原始的数据，第二行是处理后的数据，很明显，遇到连续出现6次的“93”时，RLE算法将它处理成了3个字节：“0”“6”“93”。其中“0”是标记字节，而“6”跟“93”的含义想必大家已经知道了，“6”是重复次数，“93”是符号本身。
